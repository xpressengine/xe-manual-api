FrontEndFileHandler::loadFile (   $args)

Load front end file

The $args is use as below. File type(js, css) is detected by file extension.

case js
        $args[0]: file name
        $args[1]: type (head | body)
        $args[2]: target IE
        $args[3]: index
case css
        $args[0]: file name
