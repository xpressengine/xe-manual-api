\subsection{Detailed Description}
Encode/decode Internationalized Domain Names.

The class allows to convert internationalized domain names (see R\-F\-C 3490 for details) as they can be used with various registries worldwide to be translated between their original (localized) form and their encoded form as it will be used in the D\-N\-S (Domain Name System).

The class provides two public methods, \hyperlink{classidna__convert_ae6e83c86d02d1ad9b8a7eb48435863bd}{encode()} and \hyperlink{classidna__convert_a845094b11b91976cea00bd9b9e022cd5}{decode()}, which do exactly what you would expect them to do. You are allowed to use complete domain names, simple strings and complete email addresses as well. That means, that you might use any of the following notations\-:


\begin{DoxyItemize}
\item www.\-n√∂rgler.\-com
\item xn--nrgler-\/wxa
\item xn--brse-\/5qa.\-xn--knrz-\/1ra.\-info
\end{DoxyItemize}

Unicode input might be given as either U\-T\-F-\/8 string, U\-C\-S-\/4 string or U\-C\-S-\/4 array. Unicode output is available in the same formats. You can select your preferred format via \hyperlink{}{set\-\_\-paramter()}.

A\-C\-E input and output is always expected to be A\-S\-C\-I\-I.
