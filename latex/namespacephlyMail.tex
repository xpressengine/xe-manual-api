\subsection{Detailed Description}
U\-C\-T\-C -\/ The Unicode Transcoder

Converts between various flavours of Unicode representations like U\-C\-S-\/4 or U\-T\-F-\/8 Supported schemes\-:
\begin{DoxyItemize}
\item U\-C\-S-\/4 Little Endian / Big Endian / Array (partially)
\item U\-T\-F-\/16 Little Endian / Big Endian (not yet)
\item U\-T\-F-\/8
\item U\-T\-F-\/7
\item U\-T\-F-\/7 I\-M\-A\-P (modified U\-T\-F-\/7)
