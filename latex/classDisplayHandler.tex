\subsection{Detailed Description}
\begin{DoxyAuthor}{Author}
N\-A\-V\-E\-R (\href{mailto:developers@xpressengine.com}{\tt developers@xpressengine.\-com}) \hyperlink{classDisplayHandler}{Display\-Handler} is responsible for displaying the execution result. \par
 Depending on the request type, it can display either H\-T\-M\-L or X\-M\-L content.\par
 Xml content is simple xml presentation of variables in o\-Module while html content is the combination of the variables of o\-Modue and template files/.
